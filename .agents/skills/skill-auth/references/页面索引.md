# 页面索引

本文件记录“用户与权限管理”项目中的流程、页面位置和 Pixso 地址。

## 使用说明

当用户查询某个流程或页面时，优先根据流程名称、页面名称和查询关键词匹配。

版本判断规则：

* 被绿色透明遮罩覆盖的内容是原始稿，不作为当前方案。
* 原始稿中未被绿色透明遮罩覆盖的部分，表示该部分逻辑未调整，仍然有效。
* 新稿替换的部分以新稿为准。
* 设计稿中的“岗位”是历史残留，当前统一为角色下的动态模板。
* 当前阶段不纳入尚未最终确定的二次授权逻辑。
